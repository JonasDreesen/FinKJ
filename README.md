# FinKJ — Budget Planner

Een privé budget-planner webapp voor 2 gebruikers, gebouwd met React + Vite + Tailwind, Supabase (auth + database) en gehost op GitHub Pages.

## Aan de slag

Zie [`docs/SETUP.md`](docs/SETUP.md) voor de volledige setup-instructies (Supabase account, API-sleutels, GitHub secrets, Pages).

```bash
npm install
npm run dev
```

## Status

**Fase 1 — Fundament**: login (2 vaste gebruikers, geen registratie), navigatie-skelet, Supabase-koppeling.
